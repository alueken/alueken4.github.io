Project 1: Road Rush

In Road Rush, players use arrow keys to control a car driving down a busy road. The game keeps the car within boundaries and increases the score the longer you avoid hitting other cars. It simulates traffic by randomly generating cars and features real-time user input with smooth, responsive controls.

Concepts Learned: for loops, object creation with Rect, random positioning, input handling with onKeyHold(), collision avoidance, and movement constraints.

for i in range(5):
    cars.add(Rect(random.randrange(400, 2000),

def onKeyHold(keys):

        car.centerY -= cars.speed
    if 'left' in keys and car.left > 0:
        car.centerX -= cars.speed    
    if 'right' in keys and car.right < 400:
        car.centerX += cars.speed    
    if 'down' in keys and car.bottom < 350:
        car.centerY += cars.speed
